======================================
    김헌터 봇 관리자 사용법
======================================

1. 실행 방법:
   - "봇관리자.bat" 파일을 더블클릭

2. 메뉴 설명:
   [1] 테스트봇 시작 - 개발용 봇 실행
   [2] 테스트봇 중지 - 개발용 봇 종료
   [3] 프로덕션봇 시작 - 본서버 봇 실행
   [4] 프로덕션봇 중지 - 본서버 봇 종료
   [5] 코드 배포 - 테스트 완료된 코드를 본서버로 배포
   [6] 모든 봇 재시작 - 두 봇 모두 재시작
   [7] 로그 보기 - 배포 기록 확인
   [9] 종료 - 프로그램 종료

3. 작업 순서:
   1) 테스트봇에서 개발/테스트
   2) 안정화되면 [5]번으로 배포
   3) 프로덕션봇 자동 재시작됨

4. 주의사항:
   - 봇 실행 중 창을 닫지 마세요
   - 종료는 반드시 [9]번 메뉴 사용
